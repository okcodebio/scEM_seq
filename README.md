# scEM_seq
## Description
Code for single-cell DNA methylation data analysis

<img src="https://github.com/okcodebio/scEM_seq/blob/main/scEM_seq_framework_upload.png" width="50%" height="auto" alt="scEM_seq Framework">

## Installation

```bash
git clone https://github.com/okcodebio/scEM_seq.git
```

## Dependencies

- Python >= 3.8
- NumPy >= 1.21.0
- pandas >= 1.3.0
- scikit-learn >= 0.24.0
- matplotlib >= 3.4.0
- seaborn >= 0.11.0

## Usage

```bash
cd scEM_seq
bash trim_and_qc
```
